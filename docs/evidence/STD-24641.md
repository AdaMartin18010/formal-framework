## 1. 基本信息

- **标准编号**：ISO/IEC/IEEE 24641
- **名称**：Systems and software engineering — Methods and tools for model-based systems and software engineering (MBSSE)
- **版本/日期**：2023
- **范围**：定义模型驱动系统工程（MBSSE）的术语、参考模型、方法与工具能力；涵盖模型创建、转换、验证与生命周期中的模型使用。
- **与本框架映射**：L2/L3 模型驱动、[theory-enhancement](../formal-model/theory-enhancement/)（模型驱动过程与工具能力）；formal-model 各 theory/dsl-draft 与 24641 的「模型」与「工具能力」对应。

## 2. 摘要

- 24641 提供 MBSSE 的通用参考模型与工具能力维度（如模型编辑、转换、验证、协作），与本框架 L2 元模型 → L3 标准模型 → L4 行业模型的递归建模、theory + dsl-draft 双轨及 formal-model 中的验证/转换工具对应。本框架的「知识规范与模型设计」定位与 24641 的模型驱动过程一致；不替代具体实现，与 24641 中「方法与实践」的适用范围对齐。
- 关键对齐点：MBSSE 参考模型 ↔ L2/L3 分层与递归扩展；工具能力（验证、转换）↔ formal-model/verification-tools、theory-enhancement。

## 3. 对齐维度

- **术语对齐**：model-based systems/software engineering ↔ 本框架「模型驱动」「形式化模型」；model transformation ↔ L2/L3 模型转换与 DSL 解析。
- **结构/接口**：24641 的模型类型与生命周期阶段与 L2 元模型、L3 标准模型及 theory/dsl-draft 产物对应；工具能力与 [verification-tools](../formal-model/verification-tools.md)、[alignment-L2-L3-matrix](../formal-model/alignment-L2-L3-matrix.md) 对应。
- **约束/不变式**：模型一致性、可追溯性与 L2/L3 不变式及 QUALITY_STANDARDS 中的完成度要求一致。
- **度量/指标**：模型覆盖率、转换正确性等与 L3_D08 验证与 theory-enhancement 中的验证框架对应。

## 4. 映射

- **L2**：L2 各元模型（模型驱动视角）；[formal-model](../formal-model/) 核心概念。
- **L3**：L3 各标准模型；[theory-enhancement](../formal-model/theory-enhancement/)、[verification-tools](../formal-model/verification-tools.md)。
- **L4**：行业模型与 MBSSE 实践；[industry-model](../industry-model/)。
