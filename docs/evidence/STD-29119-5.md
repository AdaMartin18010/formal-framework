## 1. 基本信息

- **标准编号**：IEEE/ISO 29119-5
- **名称**：Software and systems engineering — Software testing — Part 5: Keyword-Driven Testing
- **版本/日期**：2024
- **范围**：定义关键词驱动测试（keyword-driven testing）的术语、概念模型、测试框架与互操作性要求；与 ISO/IEC/IEEE 29119 系列其他部分（测试过程、测试文档等）协同。
- **与本框架映射**：L3_D08 测试标准模型（测试用例、测试框架、关键字/可复用测试步骤、互操作）。

## 2. 摘要

- 29119-5 将关键词驱动测试标准化为「关键字 + 参数 + 可复用步骤」的测试设计方式，与本框架 L3_D08 中的测试用例模型、测试步骤抽象、断言与数据驱动测试对应。关键词驱动便于非编程人员参与测试设计并与自动化执行互操作，对应 L3_D08 的测试框架与质量门禁集成。
- 关键对齐点：关键词/关键字 ↔ L3_D08 测试步骤与参数化用例；测试框架与互操作 ↔ L3_D09 CI/CD 中的测试阶段与工具链集成。

## 3. 对齐维度

- **术语对齐**：keyword-driven testing ↔ 本框架「关键词/关键字驱动测试」；test framework ↔ L3_D08 测试框架与流水线集成。
- **结构/接口**：29119-5 的测试资产（关键字、数据表、脚本）与 L3_D08 的 TestCase、Assertion、Coverage 等对象可对照；与 L3_D09 阶段/触发/门禁对应。
- **约束/不变式**：可追溯性、可重复执行与 L3_D08 的 Determinism、RegressionSafety 等不变式一致。
- **度量/指标**：关键字覆盖率、步骤复用率与 L3_D08 覆盖率门禁可结合使用。
