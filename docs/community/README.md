## 👥 社区结构

### 角色定义

- **项目维护者**：项目整体管理
- **核心团队成员**：技术架构和代码审查
- **领域专家**：特定领域指导
- **贡献者**：代码和文档贡献
- **社区成员**：使用反馈和问题报告
